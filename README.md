# AviUtl2 フィルタ分離プラグイン
オブジェクトに付いているフィルタ効果群を個別オブジェクトに分離するAviUtl2 プラグインです。

## 導入方法
https://github.com/Garech-mas/AviUtl2-SplitFilters/releases/latest

上記のリンクにアクセスした後、`SplitFilters-v1.xx.zip` からダウンロードできます。

ZIPの中に入っている `SplitFilters.aux2` を、AviUtl2のプレビュー画面にドラッグ＆ドロップすることで導入できます。

## 注意点
- 必ず最新バージョンのAviUtl2を導入してください。（beta22で正常動作を確認しています。）
<br>**beta21 以前ではプラグインが表示されない・認識されない場合があります。**

- Visual C++ 再頒布可能パッケージ 2015-2022 (x86) をインストールしている必要があります。

（下記リンクの 2015-2022 のリンクから確実にダウンロードすることができます。）
https://scrapbox.io/aviutl/Visual_C++_%E5%86%8D%E9%A0%92%E5%B8%83%E5%8F%AF%E8%83%BD%E3%83%91%E3%83%83%E3%82%B1%E3%83%BC%E3%82%B8

- **AviUtl (初代) では利用できません。** また、初代用「フィルタ分離」の作成予定もありません。

## 機能一覧
### フィルタ分離
- オブジェクトを右クリック → `プラグイン` → `フィルタ分離` で適用できます。
- オブジェクトに適用されているすべてのフィルタ効果を、ひとつのフィルタ効果オブジェクトとして移し替えます。

### フィルタ分離（グループ制御）
- オブジェクトを右クリック → `プラグイン` → `フィルタ分離（グループ制御）` で適用できます。
- オブジェクトに適用されているすべてのフィルタ効果を、ひとつのグループ制御オブジェクトとして移し替えます。
